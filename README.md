# SkillSwap Platform

SkillSwap is a full-stack web application developed to help users connect with people who want to teach or learn different skills. The main idea behind this project is to create a platform where users can exchange knowledge based on their expertise and interests.

Users can create profiles, add their skills, search for tutors, send skill requests, and communicate with each other using a real-time messaging system.

This project was developed using Java and Spring Boot for backend services and HTML, CSS, and JavaScript for frontend development.

---

# Features

- User registration and login
- Secure authentication and authorization using Spring Security
- User profile management
- Add and manage skills
- Search users based on skills
- Send and manage skill requests
- Real-time messaging using WebSocket
- Responsive user interface
- RESTful API integration
- Backend validation and exception handling
